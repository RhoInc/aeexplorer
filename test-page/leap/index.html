LEAP
This table summarizes adverse events from the LEAP trial by system organ class and preferred term. The table displays three columns of adverse event rates: one for Peanut Avoidance, one for Peanut Consumption, and one for both treatments. The display is fully interactive:
  • Collapse/expand nested data rows using the + and -.
  • Filter the table by adverse event prevalence.
  • Search for adverse event system organ classes and preferred terms of interest.
  • Filter the data by adverse event outcome, treatment requirement, relation to treatment, and seriousness. Hold "control" or "shift" and click to select multiple options.
More information about this tool can be found on GitHub.
