AE Explorer
LEAP
This chart shows adverse events from the LEAP trial. The chart shows two columns of data with adverse event rates, one for each treatment of the study. You can interact with the display in 3 ways:
  1. Collapse/expand nested data rows using the + and -
  2. Filter data by prevalence, AE severity and AE relation to treatment. Hold "control" or "shift" and click to select multiple options
  3. Search terms of interest.
More information about this adverse event explorer tool can be found on github.
